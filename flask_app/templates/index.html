API Reference

Expose risk predictions and explainability programmatically via our unified REST API endpoint.

POST /api/predict

Example Curl Request

curl -X POST http://127.0.0.1:5000/api/predict \
  -H "Content-Type: application/json" \
  -d '{
    "age": 55, "fasting_glucose": 145.0, "waist_cm": 98.0,
    "bmi": 30.2, "systolic_bp": 138.0, "acr": 42.0,
    "sex": 1, "total_cholesterol": 210.0, "urine_albumin": 18.0,
    "alt": 45.0, "ast": 38.0, "ggt": 52.0,
    "alp": 88.0, "total_bilirubin": 0.9, "serum_albumin": 4.0
  }'

Request Fields Specification

Field Type Valid Range Unit
age int 18 – 85 years
fasting_glucose float 50 – 500 mg/dL
waist_cm float 50 – 200 cm
bmi float 10 – 70 kg/m²
systolic_bp float 70 – 220 mmHg
acr float 0 – 5000 mg/g
sex int 1 = Male, 2 = Female enum
total_cholesterol float 50 – 500 mg/dL
urine_albumin float 0 – 500 µg/mL
alt float 0 – 500 U/L
ast float 0 – 500 U/L
ggt float 0 – 500 U/L
alp float 0 – 500 U/L
total_bilirubin float 0 – 30 mg/dL
serum_albumin float 1 – 6 g/dL
